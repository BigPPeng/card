可以看到，最终结果从1-9依次变化，所以想到要对9求余数

代码逻辑：

1.输入数字为0时，输出为0
2.输入数字是9的倍数时，输出为9
3.输入数字非0且不是9的倍数时，输出为（x % 9）

```java
class Solution {
    public int addDigits(int num) {
        if(num == 0) return 0;
        
        return num % 9 == 0 ? 9 : num % 9;
    }
}
```
